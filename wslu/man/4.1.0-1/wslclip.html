NAME

wslgsu - Component of WSL Utilities

SYNOSIS

wslclip [-hvg]

wslclip CONTENT

DESCRIPTION

This tool allows you to get or set clipboard content on Windows without X/wayland support.

OPTIONS

-h, --help

Print a simple help.

-v, --version

Print current version.

-g, --get

Get clipboard content from Windows.
